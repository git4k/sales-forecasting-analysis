## 📋 Executive Summary

A comprehensive end-to-end data analytics project analyzing 50,000+ sales transactions to uncover business insights and forecast future revenue. This project demonstrates advanced skills in data cleaning, SQL analysis, machine learning forecasting, and interactive dashboard development.

**Business Impact:** Identified revenue trends, optimized discount strategies, and built ML models achieving 95%+ accuracy in monthly revenue forecasting.

**Skills Demonstrated:**
- **Data Analysis:** Python (Pandas, NumPy), Statistical Analysis, Time Series Analysis
- **SQL:** Complex Queries, Window Functions, CTEs, Aggregations
- **Machine Learning:** Linear Regression, Random Forest, Hyperparameter Tuning
- **Visualization:** Power BI, Matplotlib, Seaborn, DAX Functions
- **Business Intelligence:** KPI Development, Trend Analysis, Forecasting

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run notebooks
jupyter notebook notebooks/01_data_cleaning_eda.ipynb
jupyter notebook notebooks/02_ml_forecasting.ipynb

# Run SQL queries
sqlite3 superstore.db < sql/sales_queries.sql
```
