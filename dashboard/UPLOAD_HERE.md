# 📸 Upload Your Power BI Dashboard Screenshot Here

## Instructions

1. Create your Power BI dashboard
2. Take a screenshot (full dashboard view)
3. Save it as: `dashboard_screenshot.png`
4. Place it in this folder (`dashboard/`)

## Screenshot Requirements

- **File name:** `dashboard_screenshot.png` (exactly)
- **Format:** PNG
- **Resolution:** 1920x1080 or higher
- **Content:** Full dashboard with all visuals

## After Uploading

The screenshot will automatically appear in the README.md under "Power BI Dashboard" section.

Push to GitHub:
```bash
git add dashboard/dashboard_screenshot.png
git commit -m "Add Power BI dashboard screenshot"
git push
```
